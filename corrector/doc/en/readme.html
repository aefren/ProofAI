ProofAI

Version: 0.2.0

Author: Alfred

Description

ProofAI is an NVDA add-on that proofreads the text selected in an edit field using any OpenAI-compatible API and replaces the selection with the improved result.

Requirements

  • NVDA 2025.3.2 or later.
  • An API key for an OpenAI-compatible service (OpenAI, OpenRouter, a local server, etc.).

Installation

  1. Download the ProofAI-0.2.0.nvda-addon file.
  2. Open the file (it will open with NVDA).
  3. Follow the on-screen instructions to install the add-on.
  4. Restart NVDA when prompted.

Configuration

  1. Go to NVDA menu > Preferences > Settings > ProofAI.
  2. Enter your API key.
  3. Set the endpoint URL (default: https://api.openai.com/v1/chat/completions).
  4. Set the model (default: gpt-4.1-mini).
  5. Optionally customize the system prompt used to instruct the model.
  6. Click OK to save.

Usage

  1. Move focus to an edit field.
  2. Select the text you want to proofread.
  3. Press NVDA+Shift+X.
  4. The add-on will send the selected text to the configured API and replace it with the corrected version.

Keyboard Shortcuts

Shortcut Action
NVDA+Shift+X Proofread and replace the selected text.

The gesture can be changed in NVDA under Preferences > Input Gestures.

Notes

  • Only the selected text is sent to the API — nothing else.
  • Compatible with any OpenAI-compatible service: OpenAI, OpenRouter, LM Studio, Ollama, etc.
  • The system prompt can be customized in the settings panel to change the correction style.
