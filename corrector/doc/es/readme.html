ProofAI

Versión: 0.2.0

Autor: Alfred

Descripción

ProofAI es un complemento para NVDA que revisa el texto seleccionado en un cuadro de edición usando cualquier API compatible con OpenAI y reemplaza la selección con el resultado mejorado.

Requisitos

  • NVDA 2025.3.2 o posterior.
  • Una clave de API para un servicio compatible con OpenAI (OpenAI, OpenRouter, un servidor local, etc.).

Instalación

  1. Descarga el archivo ProofAI-0.2.0.nvda-addon.
  2. Abre el archivo (se abrirá con NVDA).
  3. Sigue las instrucciones en pantalla para instalar el complemento.
  4. Reinicia NVDA cuando se te indique.

Configuración

  1. Ve a Menú NVDA > Preferencias > Configuración > ProofAI.
  2. Introduce tu clave de API.
  3. Establece la URL del endpoint (por defecto: https://api.openai.com/v1/chat/completions).
  4. Establece el modelo (por defecto: gpt-4.1-mini).
  5. Opcionalmente, personaliza el prompt del sistema que se usa para instruir al modelo.
  6. Haz clic en Aceptar para guardar.

Uso

  1. Sitúa el foco en un cuadro de edición.
  2. Selecciona el texto que quieras revisar.
  3. Pulsa NVDA+Shift+X.
  4. El complemento enviará el texto seleccionado a la API configurada y lo reemplazará por la versión corregida.

Atajos de teclado

Atajo Acción
NVDA+Shift+X Revisar y reemplazar el texto seleccionado.

El gesto puede cambiarse desde NVDA en Preferencias > Gestos de entrada.

Notas

  • Solo se envía a la API el texto seleccionado, nada más.
  • Compatible con cualquier servicio compatible con OpenAI: OpenAI, OpenRouter, LM Studio, Ollama, etc.
  • El prompt del sistema puede personalizarse en el panel de configuración para cambiar el estilo de corrección.
